ll cli tool

## install

安装

```shell
# dev
npm i -D ssh-switch

# global
npm i -g ssh-switch
```

## cli

```shell
# 帮助
sshs
sshs -h

# 添加
sshs add

# 列出
sshs list

# 移除
sshs remove
```
